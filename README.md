# Docker Project

A project for containerizing applications using Docker and deploying them with Docker Compose.

## 📌 Features

- Containerizes multiple apps
- Uses Docker Compose to manage services
- Simplifies deployment and scaling

## 🛠️ Technologies Used

- Docker
- Docker Compose
- Bash
